# icon-lexicon.v5.md
Semantic emoji lexicon for architecture documentation, diagrams, and technical writing.

Icons function as **semantic anchors** that help readers scan technical documents quickly.

Designed for:

- Markdown documentation
- Architecture diagrams
- draw.io / diagrams.net
- Mermaid diagrams
- Jupyter notebooks
- Technical blog posts
