# рҹ§ӯ Design Principles

- Icons are **semantic, not decorative**
- Prefer **one canonical meaning per icon**
- Allow **composability** (e.g. вҳҒпёҸ + вҡҷпёҸ = вҖңmachine in cloudвҖқ)
- Separate:
  - **Artifact vs Action vs State**
- Optimize for:
  - diagrams
  - markdown
  - infographics
